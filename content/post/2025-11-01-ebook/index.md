## Replace using regex

Use Replace with regex function in editor

```
# from this
([第一二三四五六七八九十百〇零壹贰参肆伍陆柒捌玖拾佰壹贰叁肆伍陆柒捌玖拾佰仟初序1234567890]{1,9}章.*)
# to this
## $1
```

Change the encoding to UTF-8
