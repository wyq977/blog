Conda Homebrew

Apr 03, 2022

Problems

Miniforge installed via homebrew will remove all the env upon update/uninstall, discussion.

Solution

Specify conda env in condarc

condarc example: https://docs.conda.io/projects/conda/en/latest/user-guide/configuration/sample-condarc.html

touch ~/.condarc
mkdir ~/.conda-envs

envs_dirs:
  - ~/.conda-envs
